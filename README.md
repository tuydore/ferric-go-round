Usage: ferric-go-round [OPTIONS] -n <NUM_PANELS> <FILEPATH>

Arguments:
  <FILEPATH>  Target file

Options:
  -n <NUM_PANELS>                      Number of panels to split image into
  -q <PANEL_HEIGHT_PX>                 Target image height in pixels. If not given, will preserve current image height
  -s <SIGMA>                           Gaussian blur radius as fraction of panel height [default: 0.01]
  -b <THUMBNAIL_BORDER_RADIUS>         Radius of border around image [default: 1]
  -c <THUMBNAIL_BORDER_COLOR>          Border color, given as "R,G,B" (e.g. 255,255,0). Defaults to black
  -f <THUMBNAIL_FRAC_PANEL_WIDTH>      Ratio of thumbnail width to panel width (including border) [default: 0.8]
  -h, --help                           Print help
  -V, --version                        Print version
